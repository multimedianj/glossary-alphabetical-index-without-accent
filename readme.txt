A plugin that provides you a shortcode for Glossary by Codeat to display alphabetical index without the characters with accent (ex.: À, È, É, Î, ...).

== Installation ==

1. In the WordPress Dashboard go to Plugins
1. Click the Add Plugins button and search the WordPress Plugins Directory.
1. Alternatively, you can download the plugin from [here](https://njohnson.pro/plugins/aiwa-glossary) and upload manually.
1. Activate the plugin through the 'Plugins' screen in WordPress.

== How it works ==

Just add the shortcode [aiwa-list] wherever you want.
