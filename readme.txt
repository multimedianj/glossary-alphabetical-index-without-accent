# Alphabetical Index Without Accent for Glossary by Codeat

A plugin that provides you shortcode for Glossary by Codeat to display alphabetical index without the characters with accent (ex.: À, È, É, Î, ...).

## Installation

* In the WordPress Dashboard go to Plugins
* Click the Add Plugins button and search the WordPress Plugins Directory.
* Alternatively, you can download the plugin from [here](https://njohnson.pro/plugins/aiwa-glossary) and upload manually.
* Activate the plugin through the 'Plugins' screen in WordPress.

## How it works

Just add the shortcode [aiwa-list] wherever you want.
